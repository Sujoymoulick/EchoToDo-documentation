# 🎙️ EchoToDo

> Speak. Organize. Get Things Done.

EchoToDo is a voice-powered Chrome extension that converts spoken notes into actionable tasks instantly and organizes them per website for better productivity.

---

## ✨ Features

- 🎤 Convert voice into tasks instantly
- 🌐 Website-specific task organization
- 📌 Automatic task detection from speech
- ✅ Mark tasks as completed
- 📊 Productivity tracking
- 🔔 Reminder notifications
- 📁 Local browser storage
- 📑 Side panel support
- 🌙 Dark / Light mode
- ⚡ Fast and lightweight
- 🔒 Privacy-focused
- 🚫 No API keys required

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Chrome Extension API
- Web Speech API
- Chrome Storage API

## 🚀 Installation

### Clone repository

```bash
git clone https://github.com/Sujoymoulick/EchoToDo.git
```

### Open Chrome Extensions

Go to:

```bash
chrome://extensions/
```

### Enable Developer Mode

Turn on:

```bash
Developer Mode
```

### Load extension

Click:

```bash
Load unpacked
```

Select the project folder:

```bash
EchoToDo
```

## 🎯 How It Works

### Step 1

Open EchoToDo from the Chrome toolbar.

### Step 2

Press the microphone button.

### Step 3

Speak naturally:

Example:

> "Finish login page and push code tomorrow"

### Step 4

EchoToDo converts it into:

```text
✓ Finish login page

✓ Push code tomorrow
```

### Step 5

Tasks are automatically saved for the current website.
