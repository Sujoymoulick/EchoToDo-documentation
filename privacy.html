Privacy Policy

EchoToDo is built entirely on-device. Your tasks, voice notes, and browsing context never leave your computer.

Full Disclosure

1 What Data We Collect

We collect absolutely no user data. EchoToDo has no backend, no analytics library, and no external servers. The extension cannot transmit data off your device because it has no mechanism to do so.

Specifically, we do not collect:

  • Voice recordings or audio files
  • Task text, titles, or content
  • Browsing history, URLs visited, or tab content
  • Personal identifiers (name, email, IP address)
  • Usage statistics, click events, or session data

2 How Your Data Is Stored

All data created by EchoToDo is stored exclusively in your browser's local storage sandbox using the chrome.storage.local API. This data:

  • Lives entirely on your device, inside your Chrome profile
  • Is never synced to Google servers (we do not use chrome.storage.sync)
  • Is fully deleted when you uninstall the extension
  • Can be exported at any time as a JSON file from within the extension

3 Extension Permissions Explained

EchoToDo requests the minimum permissions needed to function. Here is an exact explanation of how each permission is used:

Permission Why it's needed
storage Saves your task lists, Pomodoro timer state, and settings in local browser storage. No data is synced externally.
tabs Reads the hostname of the active tab (e.g. github.com) to automatically group task lists by website. Only the domain name is read — never the full URL, page content, or query parameters.
sidePanel Renders the EchoToDo task panel alongside any webpage so your list stays visible while you browse.
notifications Sends a desktop notification when your Pomodoro focus or break timer session ends. Notifications are never sent at any other time.
alarms Schedules a background alarm so the timer tracks time accurately even when the extension popup is closed.
offscreen Creates a hidden document to play the timer completion MP3 sound. Audio cannot be played directly from the service worker — this is Chrome's required workaround.
contextMenus Adds a right-click "Add Voice Task" menu item that opens the side panel for quick access from any webpage.
microphone Required only when you tap the mic button to record a voice note. Access is requested at that moment — not on install. Voice transcription runs entirely through Chrome's built-in Web Speech API and is never sent to any server.

4 Microphone & Voice Processing

Voice transcription is performed by the browser's native Web Speech API (SpeechRecognition). EchoToDo does not record, buffer, upload, or store any audio. The mic button is entirely optional — you can add tasks manually via the ➕ button at any time without granting microphone access.

5 Third-Party Services

EchoToDo uses no third-party services of any kind — no analytics (Google Analytics, Mixpanel, etc.), no crash reporting, no A/B testing, no advertising SDKs, and no external APIs. The extension operates entirely within Chrome's sandboxed extension runtime.

6 Open Source

EchoToDo is open source. Every line of code is publicly auditable on GitHub. You can verify directly that no data collection code exists anywhere in the codebase.
